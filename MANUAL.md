## 1. アプリの概要

このアプリは、資格試験の学習を効率的にサポートするためのWebアプリです。

| 機能 | 内容 |
|------|------|
| 対応試験 | FP&Aクラスﾞ A（追加登録も可能） |
| 利用端末 | PC・スマートフォン・タブレット（ブラウザがあれば可） |
| データ保存 | クラウド（Googleアカウントに紐づいて保存） |
| 複数端末対応 | 別のPCやスマートフォンからでも同じ成績を参照できます |

---

## 2. アクセス方法・ログイン

### 手順

1. ブラウザで以下のURLを開きます  
   **https://zakkiy-cell.github.io/study-app/**

2. ログイン画面が表示されます

   ```
   ┌─────────────────────────┐
   │     📚 学習アプリ        │
   │                         │
   │  [ Googleでサインイン ]  │
   └─────────────────────────┘
   ```

3. **「Googleでサインイン」** ボタンをクリックします

4. Googleアカウントの選択画面が表示されるので、ご自身のアカウントを選択します

5. ログインが完了すると、問題集の一覧画面（ライブラリ）が表示されます

> **ポイント：** 一度ログインすると、次回以降は自動的にログインした状態になります。
> ログアウトする場合は、ヘッダー右上のアイコンからサインアウトできます。

## 3. 問題集を選ぶ

ログイン後に表示されるのが「ライブラリ（問題集一覧）」画面です。

```
┌──────────────────────────────────────┐
│ 📚 ライブラリ         [ + 登録 ]      │
├──────────────────────────────────────┤
│ ┌────────────────────────────────┐   │
│ │ FP&A クラスA                 → │   │
│ │ 経営企画スキル検定 クラスA      │   │
│ │ 🗂 65問  📝 3回受験  ✓ 72%    │   │
│ └────────────────────────────────┘   │
└──────────────────────────────────────┘
```

- **カード**をクリックすると、その問題集の詳細画面に進みます
- バッジの見方：
  - `🗂 65問` ：問題数
  - `📝 3回受験` ：これまでの受験回数
  - `✓ 72%` ：平均正答率
  - `⚡ 苦手5問` ：正答率50%未満の問題数

## 4. 出題数を設定する

問題集カードをクリックすると、詳細画面が表示されます。
ここで出題数を選択します。

```
┌───────────────────────────────┐
│ 出題数                        │
│ [ 10問 ][ 20問 ][●30問]      │
│ [ 50問 ][ 100問 ]             │
│                               │
│ カスタム: [ 30 ] 問           │
└───────────────────────────────┘
```

| ボタン | おすすめの使い方 |
|--------|----------------|
| 10問 | 隙間時間の5分練習 |
| 20問 | 朝のウォームアップ |
| **30問（デフォルト）** | **通常の学習セッション** |
| 50問 | 集中学習 |
| 100問 | 本番想定の模擬試験 |

カスタム入力欄に直接数字を入力することで、任意の問題数も指定できます。

## 5. 学習モードを選ぶ

出題数の下にある3つのボタンから学習モードを選びます。

### 模擬試験モード（おすすめ）

```
[ 🎯 模擬試験を開始 ]
```

- すべての章からランダムに出題されます
- 本番試験に近い形式で力試しができます
- **まずはこのモードからお試しください**

### 分野別学習

```
[ 📖 分野別学習 ]
```

- 特定の章（カテゴリ）に絞って学習できます
- クリックすると章の一覧が表示され、学習したい章を選択します
- 苦手な章を集中的に練習したいときに使います

### 苦手問題モード

```
[ ⚡ 苦手問題 ]
```

- 過去に間違えた問題や正答率の低い問題が優先的に出題されます
- 受験履歴が蓄積してくると、より効果的に活用できます
- 受験回数が少ない場合は「まだデータが足りません」と表示されます

## 6. 問題を解く

### 画面の見方

```
┌─────────────────────────────────────────┐
│ 模擬試験   3/30           ⏱ 45:22      │
│ ████████░░░░░░░░░░░░░░░░░░░ 10%         │
├─────────────────────────────────────────┤
│ 第3章 予算管理                           │
│                                         │
│ 予算編成において、ローリング予算の       │
│ 特徴として最も適切なものはどれか。       │
│                                         │
│ ┌─────────────────────────────────┐     │
│ │ A  一定期間の予算を固定する     │     │
│ ├─────────────────────────────────┤     │
│ │ B  期末に向けて予算を見直す     │     │
│ ├─────────────────────────────────┤     │
│ │ C  常に一定の将来期間分を維持   │     │  ← 選択すると正解/不正解が表示
│ ├─────────────────────────────────┤     │
│ │ D  年間を通じて月次で修正する   │     │
│ └─────────────────────────────────┘     │
└─────────────────────────────────────────┘
```

| 表示 | 内容 |
|------|------|
| `3/30` | 現在の問題番号 / 全問題数 |
| `⏱ 45:22` | 残り時間（模擬試験モードのみ） |
| プログレスバー | 進捗（問題を解くごとに伸びる） |
| カテゴリ表示 | 問題が属する章・分野 |

### 答え方

1. A〜Dの選択肢のうち、正しいと思うものを**クリック（タップ）**します
2. 即座に正解・不正解が表示されます

   - **正解の場合：** 選択肢が緑色に変わり「正解！」と表示
   - **不正解の場合：** 選択した選択肢が赤色、正解が緑色に変わり「不正解」と表示

3. 解説が表示されます。内容をしっかり読みましょう

4. **「次の問題」** ボタンで次へ進みます

> **ポイント：** 選択肢の並び順は出題のたびにランダムに変わります。  
> 「3番目の選択肢が正解」といった位置での暗記はできないため、内容で理解することが重要です。

全問題を解き終えると、結果画面が表示されます。

```

│            72%                          │
│        22問 / 30問 正解                  │
│           [ 合格圏 ]                     │
├─────────────────────────────────────────┤
│ 分野別スコア                             │
│ FP&Aとは何か    ████████████░░░ 80%      │
│ 戦略・組織      ████████░░░░░░░ 60%      │
│ 予算管理        ██████████████░ 90%      │
│ 財務モデリング  ████████░░░░░░░ 55%      │
│ 財務予測        ██████████████░ 88%      │
│ 投資決定        ██████░░░░░░░░░ 45%      │
├─────────────────────────────────────────┤
│ [ 間違いを確認 ] [ 苦手問題を練習 ]      │
│ [ もう一度挑戦 ] [ ライブラリへ戻る ]    │
└─────────────────────────────────────────┘
```

### ボタンの使い方

| ボタン | 内容 |
|--------|------|
| 間違いを確認 | 間違えた問題の一覧と解説をまとめて確認できます |
| 苦手問題を練習 | 間違えた問題をそのまま再出題します |
| もう一度挑戦 | 同じ設定で新しいセッションを開始します |
| ライブラリへ戻る | 問題集一覧に戻ります |

### 分野別スコアの見方

| 正答率 | 表示色 | 意味 |
|--------|--------|------|
| 80%以上 | 緑 | 定着している |
| 60〜79% | 黄 | もう少し練習が必要 |
| 60%未満 | 赤 | 重点的に復習が必要 |

## 8. 成績・履歴を見る

ライブラリ画面の問題集カードから「成績を見る」ボタンをクリックすると、累積の成績が確認できます。

```
┌─────────────────────────────────────────┐
│ 成績詳細                                 │
├──────────────┬──────────────┬───────────┤
│ 総受験回数   │ 累積正答率   │ 苦手問題  │
│    12回      │    68%       │   8問     │
├─────────────────────────────────────────┤
│ 問題別成績   受験履歴                    │
├─────────────────────────────────────────┤
│ #Q1  ローリング予算とは...  85%  7回     │
│ #Q2  ゼロベース予算の特...  43%  7回  ⚡ │  ← ⚡は苦手問題
│ #Q3  予算差異分析のうち... 100%  6回     │
└─────────────────────────────────────────┘
```

- **問題別成績タブ：** 各問題の正答率と挑戦回数が確認できます
- **受験履歴タブ：** 過去のセッションごとの日時・スコアが時系列で確認できます

> **ポイント：** このデータはGoogleアカウントに紐づいて保存されるため、  
> 別の端末からログインしても同じ成績が表示されます。

## 9. 問題集を登録する（管理者向け）

> この操作は問題集を追加・管理する担当者向けです。一般ユーザーは通常不要です。

### JSONファイルで登録する

1. ライブラリ画面右上の **「+ 登録」** ボタンをクリック
2. 登録画面でJSONファイルをドラッグ＆ドロップ（またはクリックして選択）
3. 「登録しました」と表示されれば完了

### Excelファイルで登録・編集する（推奨）

Excelで問題集を作成・編集して登録できます。

#### テンプレートのダウンロード

1. 登録画面の「📄 Excelテンプレートをダウンロード」ボタンをクリック
2. `question_template.xlsx` がダウンロードされます

#### Excelファイルの構成

テンプレートは **2つのシート** で構成されています。

**① metaシート（問題集の基本情報）**

| 列名 | 内容 | 例 |
|------|------|-----|
| id | 問題集の一意ID（英数字） | `fpa_class_a` |
| name | 問題集名 | `FP&A クラスA` |
| description | 説明文 | `経営企画スキル検定...` |
| timeLimit | 制限時間（分） | `90` |
| version | バージョン | `1.0.0` |

**② questionsシート（問題データ）**

| 列名 | 内容 | 例 |
|------|------|-----|
| id | 問題番号 | `1` |
| category | カテゴリ（章名） | `第3章 予算管理` |
| question | 問題文 | `ローリング予算の特徴は...` |
| choiceA | 選択肢A | `一定期間の予算を固定する` |
| choiceB | 選択肢B | `期末に向けて見直す` |
| choiceC | 選択肢C | `常に一定の将来期間分を維持` |
| choiceD | 選択肢D | `年間を通じて修正する` |
| answer | 正解（A/B/C/Dのいずれか） | `C` |
| explanation | 解説文 | `ローリング予算とは...` |

#### Excelを登録する

1. 作成したExcelファイルを登録画面にドラッグ＆ドロップ
2. 「登録しました」と表示されれば完了

#### 既存の問題集をExcelに書き出す

1. 登録一覧で対象の問題集の **「⬇ Excel」** ボタンをクリック
2. Excelファイルがダウンロードされます（編集して再登録できます）

## 10. よくある質問

**Q. スマートフォンでも使えますか？**  
A. はい。スマートフォンのブラウザからも同様にご利用いただけます。Chrome・Safariに対応しています。

---

**Q. 別のPCからログインしたらデータはどうなりますか？**  
A. 同じGoogleアカウントでログインすれば、成績・履歴はそのまま引き継がれます。

---

**Q. 問題の選択肢の順番が変わるのはなぜですか？**  
A. 位置で答えを覚えてしまわないよう、出題のたびに選択肢の順番をランダムに入れ替えています。

---

**Q. 途中でブラウザを閉じたらどうなりますか？**  
A. 進行中のセッションデータは失われ、結果は成績に記録されません。時間に余裕のあるときに受験することをおすすめします。

---

**Q. 問題が少なく設定した数に満たない場合はどうなりますか？**  
A. 問題集の総問題数を上限として出題されます（例：65問の問題集で100問に設定した場合、65問出題されます）。

---

**Q. ログアウトしたいのですが？**  
A. 画面右上のユーザーアイコンをクリックし「サインアウト」を選択してください。

---

**Q. 問題内容に誤りを見つけた場合は？**  
A. 問題集の管理者にご連絡ください。管理者はExcelで問題を修正・再登録できます。
